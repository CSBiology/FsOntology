# FsOntology
A toolkit for working with ontology data structures in .NET

## build tasks

- Build the project:
    - If developing in VS, you can use VS interface (e.g. *rightclick -> build*)
    - via cli: `./build.cmd`

- Test the project:
    - If developing in VS, you can use VS Test explorer
    - via cli: `./build.cmd runTests`
